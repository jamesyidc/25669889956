- 📈 **见底信号+涨幅后8做多**（深绿色）

### 3. 编辑配置
点击策略卡片中的 **✏️ 铅笔图标**，会弹出配置窗口：

#### 配置项说明

**📈 RSI总和阈值**
- 含义：市场RSI总和的阈值线
- 范围：300 - 1500
- 默认：800
- 建议：
  - RSI < 800 通常表示市场超卖，适合做多
  - 可根据市场行情调整，熊市可适当降低（如700）
  - 牛市可适当提高（如900）

**💰 单币限额 (USDT)**
- 含义：每个币种最多开仓的金额
- 范围：1 - 100 USDT
- 默认：5.0 USDT
- 建议：
  - 新手：2-5 USDT（风险较低）
  - 进阶：5-10 USDT（中等风险）
  - 激进：10-20 USDT（高风险高收益）

### 4. 启用策略
在策略卡片右上角，打开 **开关** 启用策略

## 📊 策略工作原理

### 触发条件（三个条件同时满足）
1. **市场情绪** 出现：
   - ✅见底信号
   - 📉底部背离

2. **RSI总和** < 设定的阈值（默认800）

3. **自动选择币种**：
   - 策略1：从15个常用币种中选涨幅前8名
   - 策略2：从15个常用币种中选涨幅后8名

### 开仓配置
- **杠杆**: 10倍
- **总投入**: 可用余额的 1.5%
- **分配方式**: 平均分配给8个币种
- **单币上限**: 由配置决定（默认5 USDT）

### 监控机制
- **检查频率**: 每60秒检查一次
- **冷却时间**: 触发后1小时内不重复执行
- **后台进程**: bottom-signal-long-monitor（PM2托管）
